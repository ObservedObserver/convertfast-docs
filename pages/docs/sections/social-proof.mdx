# Social Proof Section

The Social Proof section showcases testimonials from your customers or users.

import { SocialProof } from '@/components/home/social-proof'

<SocialProof />

To add the Social Proof section to your landing page:

```bash
npx convertfast-ui@latest page <your-page> add social-proof
```

## Components

- **Card**: Sourced from `shadcn-ui`.
- **Avatar**: Sourced from `shadcn-ui`.
