# Call to Action (CTA) Section

The CTA section encourages users to take a specific action, such as signing up or making a purchase.

import { CTA } from '@/components/home/cta'

<CTA />

To add the CTA section to your landing page:

```bash
npx convertfast-ui@latest page <your-page> add cta
```

-  **Button**: Sourced from `shadcn-ui`.
