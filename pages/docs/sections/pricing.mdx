# Pricing Section

The Pricing section provides details about the pricing plans available for your product or service.

import { PricingSection } from '@/components/home/pricing'

<PricingSection />

To add the Pricing section to your landing page:

```bash
npx convertfast-ui@latest page <your-page> add pricing
```
